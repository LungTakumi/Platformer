# KNOCK THEM OUT! (On play store)

This is a 2D platformer game project.

-------------------------------------------------

Controls are: 

A, D for left and right. 

SPACE for jump. 

SHIFT for Attack.

CTRL for Slide.

F for Throw Attack.
